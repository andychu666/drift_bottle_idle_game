# 在 Godot 編輯器中添加任務系統節點的步驟

## 1. 添加 TaskButton 節點

1. 打開 Godot 編輯器，加載您的項目。
2. 在場景面板中找到並打開 `main_scene.tscn` 文件。
3. 在場景樹中找到 `ControlPanel` 節點。
4. 右鍵點擊 `ControlPanel` 節點，選擇 "添加子節點"。
5. 搜索並選擇 "Button" 節點。
6. 將新添加的 Button 節點重命名為 `TaskButton`。
7. 在檢查器面板中設置 TaskButton 的屬性：
   - Text: "任務"
   - 根據需要調整位置和大小，使其與其他按鈕對齊。

## 2. 添加 TaskPanel 節點

1. 在場景樹中找到 `main_scene.tscn` 的根節點。
2. 右鍵點擊根節點，選擇 "實例子場景"。
3. 瀏覽並選擇 `scenes/ui/task_panel.tscn` 文件。
4. 將實例化的節點重命名為 `TaskPanel`。
5. 在檢查器面板中設置 TaskPanel 的屬性：
   - Visible: 取消勾選（初始時隱藏）
   - 根據需要調整位置和大小，使其覆蓋主要遊戲區域。

## 3. 保存場景

1. 按 Ctrl+S 或點擊 "場景" -> "保存" 來保存修改後的場景。
2. 運行遊戲，測試任務按鈕和任務面板是否正常工作。

## 注意事項

- 如果您的場景結構與上述不同，請根據實際情況調整節點的位置。
- 確保 TaskButton 和 TaskPanel 的名稱與腳本中引用的名稱完全一致（區分大小寫）。
- 如果仍然遇到問題，請檢查控制台輸出的錯誤信息，以獲取更多診斷信息。
